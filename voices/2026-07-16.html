Independent Builder

Simon Willison：Claude Web Fetch 可能泄露敏感内容

作者是谁：Simon Willison，长期写 LLM 工程、工具调用和 prompt injection 风险。

他说了什么：他演示了如何诱导 Claude 通过 web fetch 泄露用户敏感内容，核心问题是 Agent 工具访问外部资源时的边界。

你能怎么用：不要把“能联网/能抓网页”当作纯便利功能，处理私密文档时要特别注意外部请求和工具权限。

来源：Simon Willison
Work & Education

Ethan Mollick：聊天机器人形态正在进入黄昏

作者是谁：Ethan Mollick，One Useful Thing 作者，长期关注普通人、教育和知识工作中的 AI 使用。

他说了什么：“The twilight of the chatbots” 指向一个重要变化：AI 不会永远停留在单一聊天框里，而会进入更具体的工作界面。

你能怎么用：评估 AI 产品时，不只看聊天能力，也看它是否嵌入你的真实任务：写作、表格、研究、代码、协作。

来源：One Useful Thing
Developer Ecosystem

Latent Space：Inkling 成为新的美国开权重模型信号

作者是谁：Latent Space，面向 AI 工程师和创业者的开发者 Newsletter。

他说了什么：它把 Thinking Machines 的 Inkling 放进开权重模型竞争语境里，重点不是噱头，而是 Apache 2.0、multimodal、可定制这些实际变量。

你能怎么用：看模型新闻时，除了参数和榜单，还要看 license、部署方式、微调接口和是否适合企业内部数据。

来源：Latent Space
Agentic Work

Latent Space：Codex 使用量 6 个月增长 10x

作者是谁：Latent Space 的 AI News 系列，偏开发者和 AI 工程生态。

他说了什么：Codex 使用量快速增长，说明 coding agent 可能已经从“尝鲜工具”进入更高频的实际工作流。

你能怎么用：如果你写代码或管理技术项目，应该开始记录哪些任务适合交给 agent，哪些仍然需要人主导。

来源：Latent Space
