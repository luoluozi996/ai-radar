Graphite Diamonds

Graphite 发布的 AI code review 能力，把自动审查接入 pull request 流程，用来补充团队 review，而不是替代最终责任人。

适合谁
已经在 GitHub 上协作、希望减少明显代码问题和重复 review 成本的小团队。
核验
官方发布页和产品文档可直接查看功能边界与接入方式。
限制
自动 review 不能证明业务正确性；仍要保留测试、人工复核和回滚策略。

Zed Agent Panel

Zed 的 agent 面板代表 IDE 正在把“对话、编辑、运行、审查”合并成同一工作台，适合观察下一代编辑器交互。

OpenPipe ART
