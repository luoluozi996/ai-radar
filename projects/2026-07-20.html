Qwen-Agent

Qwen 的开源 agent 组件库，覆盖函数调用、MCP、代码解释器与 RAG；既可接入云端服务，也可对接自部署模型服务。

适合谁
需要把工具调用、检索与自定义工具做进 Python 应用的开发者。
核验
README 给出 PyPI 安装、可选依赖及自定义工具示例；部署路径清晰。
限制
仍要自行承担模型服务、权限边界与任务评估，不能把框架当成可靠性保证。

OpenAgent

面向个人的开源助手项目，README 明确覆盖 RAG、电脑/浏览器使用和 coding agent，并提供只读 Demo 与可重置 Playground。

适合谁

核验

限制
