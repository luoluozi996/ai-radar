Hallmark

一个面向 Claude Code、Cursor、Codex 等 agent 的设计 skill，用结构变化、主题选择和质量门槛减少 AI 生成页面的“模板味”。

为什么重要
它不是组件库，而是把设计判断写进 agent 的行为规则，适合解决 AI 页面同质化。
适合谁
经常用 coding agent 生成页面、落地页、产品原型的人。
核验
README/技能说明与安装路径可查；来源包括 Agent Skills Library 与项目说明。
Evaluation86来源

Graphify

把代码库、文档和 schema 映射成可查询知识图谱，让 AI coding assistant 通过关系路径理解代码，而不是每次重新 grep。

为什么重要
Agent 写代码时最常卡在上下文重建；图谱把调用、导入、定义等关系显式化。
适合谁
维护中大型代码库、经常问跨文件架构问题的工程团队。
核验
Graphify docs 描述安装、构图、查询、MCP 接入和本地输出文件。
Evaluation82来源
Category Map · Score 76

awesome-llm-apps

一个收集可运行 AI Agent、RAG、MCP、语音和多 agent 模板的仓库，价值在于能直接 clone、改造、运行。

为什么重要
它把“我想做一个 LLM 应用”转成可复制的工程起点，而不是只给灵感列表。
适合谁
想快速学习 AI 应用结构、复用模板、比较不同 agent/RAG 写法的开发者。
核验
GitHub README 包含 Quick Start、模板目录、license 和示例分类。
