系统流程

这个项目的重点不是堆信息，而是把抓取、去重、筛选、发布和通知做成稳定闭环。

抓取

按时间窗口读取公开信号。

清洗

过滤重复、低质量或弱来源内容。

评分

结合新鲜度、可信度和实用性排序。

归类

拆分项目、事件、模型、观点和 X 观察。

发布

生成静态 HTML 并更新 GitHub Pages。

同步

Notion 去重写入并发送微信推送。
